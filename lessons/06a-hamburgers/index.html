This is what I’d call the normal case. Order doesn’t really matter, but I do know that I want all of my condiments together and all of my veggies together as well. Let’s make sure that as we move our burger markup around they stay together. Yep, you're burger is actually a bunch of nested ingredients. Chomp on that one for a bit.

          
  <ul>
    <li class="bun top">Pretzel Bun</li>
    <li class="condiments">
      <strong>Condiments</strong>
      <ul>
        <li class="ketchup condiment">Ketchup</li>
        <li class="mustard condiment">Dijon Mustard</li>
      </ul>
    </li>
    <li class="veggies">
      <strong>Veggies</strong>
      <ul>
        <li class="tomato veggie">Plum Tomato</li>
        <li class="lettuce veggie">Arugula</li>
      </ul>
    </li>

  </ul>
